
# A simple calculator written in C

This repository contains a basic calculator program implemented in C. The calculator can perform arithmetic operations such as addition, subtraction, multiplication, and division.



## Usage

Compile the Code:

```bash
  Compile the calculator.c file using a C compiler
```
 Run the Calculator:

 ```bash
 ./calculator
 ```   
 Input Format:
 ```bash
 The calculator prompts you to enter an operator (+, -, *, /) and two numbers.
 ```
